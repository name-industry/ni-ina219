NI_INA219

new NI_INA219()

(v.0.0.1) - WaveShare UPS Raspberry Pi Hat uses a TI INA219_sensor

Texas Instruments INA219 Zero-Drift Bi-Directional Current/Power monitor with I2c interface.

INA219 Chip Specs
WaveShare UPS Product page

This library builds on these two versions:

WaveShare's Python demo code
nodejs version by brettmarl on GitHub

Uses I2c-bus temporarily for developing - but will be removed once complete as the module should be provided a promise-based bus on instantiation.

Testing hardware:
1 x Raspberry Pi 4b with the WaveShare UPS hat installed.
2 x NCR18650B 18650 Battery

#note: Battery type Panasonic-Japan 3400mAh Li-ion 3.7V Flat Top Rechargeable [3.6 <> 4.2 cutoff 2.5]

(async) initialize(i2cAddress, busNumber, configurationTemplateId, useLogging, loggingType) → {Promise.<(ResultObject|ErrorResultObject)>}

Start register/calibrate bus & sensor

Gets a handler to the INA219 chip via I2c. On success configures the chip and runs an initial calibration to ensure correct values on the register.
