(async) setCustomCalibration(busVoltageMax, shuntResistanceOhms, gainVoltage, currentMaxExpected) → {Promise.<(ResultObject|ErrorResultObject)>}

set custom calibration values

Description:
  • Opens up the calibration register for advanced usage. You can can use the Constants file exposed to create any calibration values and have them stored on the register. This is for uses outside the Templates where you need to fine tune the measured results returned via the public interface calls. Setting these randomly will not do much as they should be properly matched to the configuration register inputs.
